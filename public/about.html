About the BU Attendance Portal

The BU Attendance Portal is a web-based platform designed to simplify the tracking and management of attendance records for organizations of all sizes.

Features

  • Secure user authentication with hashed passwords
  • Real-time attendance data fetched from an external API
  • Persistent storage of user and attendance records in a database
  • Responsive design that works on desktop, tablet, and mobile
  • Contact form for user inquiries with client-side validation

How It Works

Users register and log in to access their personal attendance dashboard. The system retrieves attendance records from the database and displays them in a clear, readable format. Administrators can manage records and respond to contact inquiries.
