About BU Attendance Portal
The BU Attendance Portal is a web-based attendance management system developed for Bugema University. It enables lecturers to generate QR codes for each lecture, and students to scan them to record their attendance in real time.
The system supports biometric fingerprint verification, GPS location checking, and role-based access for Admins, Lecturers, Students, and the QA Office.
Key Features
Attendance Tracking
  • QR code generation per lecture
  • Real-time attendance scanning
  • GPS location verification
  • Biometric fingerprint support
User Roles
  • Admin — full system control
  • Lecturer — manage courses & lectures
  • Student — enroll & scan attendance
  • QA Office — monitor & report
Course Management
  • Admin creates & assigns courses
  • Student enrollment requests
  • Admin approval workflow
  • Semester management
Technology Stack
  • Node.js + Express backend
  • SQLite database
  • HTML5 QR Scanner
  • WebAuthn biometric API
